synchro non liée

Synchroniser entre vos appareils, sans serveur

Option A — votre compte Google Drive (recommandée). Kiosque range ses données dans l’espace « données d’application » caché de votre Drive et fusionne les modifications entre tous vos appareils (PC pro, PC perso, iPhone). Nécessite que cette page soit ouverte depuis une adresse https (ex. GitHub Pages) et un identifiant OAuth gratuit — voir le guide fourni avec l’app.

Option B — fichier dans un dossier synchronisé. Si un jour vous installez « Google Drive pour ordinateur » (ou OneDrive, Dropbox…), liez un fichier kiosque-sync.json placé dans le dossier synchronisé : le service le transporte entre vos PC.

Option C — manuel, zéro configuration. Exportez un JSON, déposez-le sur drive.google.com, puis importez-le sur l’autre appareil : l’import fusionne sans rien écraser.
